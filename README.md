**Tech Stack / Frameworks / Libraries:**

- Vue 3
- MongoDB
- Mongoose
- Socket.io
- Express
- Tailwind CSS
- Vite

**Overview**

*Good Point!* is a demonstration full-stack project that uses socket.io to create chat rooms. Users can log in, add and create chat rooms, and discuss topic with others.
